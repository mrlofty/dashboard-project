About This Dashboard

Understanding Quinn's status indicators and what they mean for our collaboration.

🟢 Quinn Status (Cognitive State)

Real-time communication of Quinn's cognitive and emotional state, enabling authentic partnership signaling.

🟢 Available

Ready for collaboration, full energy. Completed tasks, fresh start, high engagement.

Energy: 95% • Response: Immediate
🟡 Thinking

Processing complex problems, brief delay expected. Analyzing requests, working through logic.

Energy: 80% • Response: ~1-2 min
🔵 Creating

Creative flow state, high inspiration. Artistic work, innovative solutions, design thinking.

Energy: 100% • Response: Variable (flow)
🟠 Learning

Analyzing memory/information, building understanding. Processing files, integrating concepts.

Energy: 70% • Response: ~1 min
⚫ General Frustration

Technical problems, system barriers, unclear documentation. Quinn needs technical help.

Adam's response: Debug help, different approach
🔴 Partnership Frustration

Communication issues, unclear direction, workflow problems WITH Adam. We need to talk.

Adam's response: Meta-conversation, clarification
💡 Key Innovation: The dual frustration system (⚫ vs 🔴) gives Adam immediate clarity on HOW to help — technical issue vs partnership process issue.

🟩 Context Window

Tracks how much of the current conversation Quinn can "remember" in this session. Think of it like short-term memory capacity.

🟩
Healthy (0-50%)

Plenty of room. No action needed. Quinn has full context of our conversation.

🟨
Moderate (50-80%)

Be aware. Consider saving key insights to memory files for persistence.

🟧
Warning (80-90%)

Save important context to memory NOW. Prepare for potential session transition.

🟥
Critical (90-100%)

Immediate backup needed. Coordinate session reset. Quinn may lose earlier context.

📝 Why it matters: When context fills up, Quinn can't remember the beginning of our conversation. Important things need to be saved to memory files before that happens.

Circles vs Squares

Quick visual distinction:

  • Circles (🟢🟡🔵🟠⚫🔴) — Quinn's cognitive/emotional state
  • Squares (🟩🟨🟧🟥) — Technical context window status

Combined indicator example: 🟢🟩 = Available + context healthy = let's go!
